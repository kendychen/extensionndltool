# NDL Tool - Mail Manager Extension

Chrome Extension for [NDL Tool](https://ndltool.com) - Mail account management.

## Features

- Auto OAuth login to Microsoft accounts
- Read mail without server resources
- Unlock Identity accounts via proxy
- Secure: only communicates with ndltool.com

## Installation

1. Download or clone this repository
2. Open Chrome → `chrome://extensions/`
3. Enable **Developer Mode** (top right toggle)
4. Click **Load unpacked** → select this folder
5. Copy the **Extension ID** shown under the extension name
6. Go to [ndltool.com](https://ndltool.com) → paste Extension ID in the connection box

## Requirements

- Google Chrome or Chromium-based browser
- Active account on [ndltool.com](https://ndltool.com)
